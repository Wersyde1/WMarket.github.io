Wersyde Маркет

Garry's Mod - это компьютерная игра, физическая песочница, которая позволяет игроку манипулировать объектами и экспериментировать с физикой, реалистичность которой обеспечивается движком Source engine. Эта модификация была создана для демонстрации обширных возможностей движка Source, который ограничивает свободу игроков только их собственным воображением.

С помощью Garry's Mod вы можете выполнять любые действия с объектами и персонажами из игр на движке Source (CS:S, CS:GO, TF2, HL2 и других игр Steam на движке Source), например, изменять текстуры, изменять размер, копировать, подключать, взрывать, крепить колеса, меняйте выражения лиц персонажей, меняйте их кости, ставьте их в причудливые позы и т.д. Модификации предоставляют игроку огромные возможности для творчества.
